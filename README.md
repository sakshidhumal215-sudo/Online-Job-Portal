# Online Job Portal

A web-based platform for job posting, job searching, and application tracking.

## Features

- User registration and secure login
- Job posting and job search functionality
- Apply for jobs and track applications
- Email notifications for updates (selection/rejection)
- Communication between recruiters and job seekers

## Technologies Used

- Python (Flask)
- SQLite
- HTML, CSS
